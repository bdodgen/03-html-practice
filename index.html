How do I see the differences between HTML and CSS?

  1. What is the difference between the usage of HTML and CSS?

    2. HTML is used for the content of a page, and CSS is used for styling the content.

  2. What is the newest version of HTML?

    3. HTML5

  3. What are the top five most used HTML5 elements? If not sure, make a guess.

    • div
    • li
    • a
    • p
    • img

  4. Have you checked out the HTML5 cheatsheet?

    5. Yes

  5. What is the difference between semantic and non-semantic elements?

    6. Semantic elements make sense to the average English-speaking human. They describe what they contain (like header, footer, section, etc.), and non-semantic elements (like div) don't.

  6. Why do we prefer using semantic elements?

    7. Because it's better to write code for humans first since we're the ones maintaining it.

  7. Do all browsers support all of the newest HTML / CSS elements?

    8. no

  8. Test the following elements on Can I Use: svg, nav, code, form, datalist. Which browser version supports them and which do not?

    • svg has full support on everything except IE, where it has partial support
    • nav is fully supported
    • code is fully supported
    • form is supported on everything except IE
    • datalist has full support on most browsers, except only partial support for IE and Firefox
